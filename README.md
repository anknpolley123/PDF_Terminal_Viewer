# PDF_Terminal_Viewer 📄🌈
​**PDF_Terminal_Viewer** is a lightweight, high-speed terminal utility designed to convert PDF documents into a vibrant, colored text experience. It is specifically optimized for Linux environments like **Kali Linux** and **Termux**, providing seamless support for both digital and scanned documents.

## Features

**Colored Output:** Uses *lolcat* for a rainbow terminal effect.

**OCR Support:** Automatically handles scanned PDFs using *ocrmypdf*.

**Language Support:** Optimized for English and Bengali text.

**Easy Setup:** One-command dependency installer.
